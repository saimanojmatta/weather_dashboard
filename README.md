# Weather Dashboard

## Overview

Weather Dashboard is a modern web application that provides real-time weather information and forecasts for the next 5 days. Leveraging the power of [OpenWeatherMap](https://openweathermap.org/), this application delivers accurate and detailed weather data using a clean and responsive UI built with the latest web technologies.

## Tech Stack

- **Next.js**: React framework for server-side rendering and generating static websites.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Shadcn UI**: Component library for beautiful and accessible UI components.
- **Recharts**: Library for building dynamic and responsive charts.
- **OpenWeatherMap API**: Service to fetch current weather and forecast data.

## Implementation and Features

- **Accurate Weather Data**: Fetches current weather and 5-day forecast data from OpenWeatherMap API.
- **Beautiful UI Components**: Utilizes Shadcn UI components for a polished and user-friendly interface.
- **Dynamic Search**: Users can type in the search box to get city suggestions and view weather details for the selected city.
- **Detailed Forecast**: Displays a line chart using Recharts to compare the minimum, maximum, and average temperatures over the next 5 days.

## Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/saimanojmatta/weather_dashboard.git
   ```
2. **Navigate to the project directory**:
   ```sh
   cd weather-dashboard
   ```
3. **Install dependencies**:
   ```sh
   npm install
   ```
   or
   ```sh
   yarn install
   ```
4. **Run the development server**:

   ```sh
   npm run dev
   ```

   or

   ```sh
   yarn run dev
   ```

5. **Open your browser and go to**: `http://localhost:3000`

## Usage

- **Search for a City**: Enter the name of the city in the search bar. Suggestions will appear as you type.
- **View Current Weather**: The current weather data for the selected city will be displayed, including temperature, humidity, and weather conditions.
- **View 5-Day Forecast**: A detailed 5-day weather forecast will be shown, including a line chart comparing the minimum, maximum, and average temperatures.
